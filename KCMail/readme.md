**KCmail**是一个自动下载邮件里附件的脚本。初衷是帮助学委收作业。。。

**KCmail**详细介绍博文[地址][addr]

---

###强制性要求

* 目前仅支持 zip 格式的压缩文件
* 目前仅支持网易邮箱，qq 邮箱不支持，gmail 未测试
* 目前脚本是一次性作用的，不支持后期有后续邮件下载到同一个文件夹中
